HP TeamSite - WCM(Web Content Management)

웹사이트의 컨텐츠를 생성, 테스트, 배포, 관리 할 수 있는 전사적 컨텐츠 관리 솔루션입니다.

HP TeamSite

HP TeamSite는 컨텐츠 생성, 테스트, 관리를 위한 협업 솔루션입니다. 고성능 오브젝트 저장소 아키텍쳐를 기반으로 하여 다양한 유형의 컨텐츠(문서, 이미지, 동영상, 사이트 레이아웃, 프로그램 코드, DB 데이터 등)와 협업을 위한 워크플로우 및 승인 프로세스를 관리하기 위한 플랫폼을 제공합니다.

주요 기능

컨텐츠 수집 및 통합 자동화

브랜치 구조에 의한 멀티 사이트 통합 관리

컨텐츠 재사용을 위한 템플릿 제공

버츄얼라이제이션: 가상 테스트 환경 제공

태스크(Task) 기반 동적 워크플로우

메타데이터 입력 및 검색

사이트 버전관리

HP OpenDeploy

HP Autonomy OpenDeploy는 기업의 분산 네트워크 환경에서 코드 및 컨텐츠 배포를 자동화하는 솔루션입니다. 트랜잭셔널 배포로 다양한 플랫폼의 대상 서버 컨텐츠를 효율적으로 동기화할 수 있도록 지원하며, 컨텐츠의 압축 및 암호화로 안전하고 신속한 배포를 책임집니다.

주요 기능

트랜잭셔널 배포

멀티 채널 / 플랫폼 배포

다단계 배포 / 동시 배포

암호화 배포 / 압축 배포

에러 시 자동 복구(Rollback)

웹브라우저 기반 관리 UI

Payload Adaptor(SCM 연동)

Delivery Adaptor(BEA, IBM, MS)

HP LiveSite

HP Autonomy LiveSite는 컴포넌트와 템플릿 기반의 웹 컨텐츠 생성 및 디스플레이를 위한 플랫폼으로 비즈니스 부서가 신속하고 비용 효율적으로 웹사이트를 퍼블리싱하고 활용할 수 있는 WYGIWYG 방식의 사이트 저작 및 퍼블리싱을 위한 Add-On 모듈입니다. IT 부서에서 제공된 공통 컴포넌트를 활용하여 해당 사업부에서 템플리팅 기반의 웹 페이지를 퍼블리싱할 수 있는 통합 플랫폼을 제공합니다.

주요 기능

WYSIWYG 방식 저작 툴

Drag & Drop 기능

컴포넌트 공유 및 재사용

템플릿을 활용한 페이지 레이아웃 자동 생성

페이지 별 레이아웃 적용
